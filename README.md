## 🎯 Sobre o Projeto

EconomizaPinhal é uma plataforma web que conecta consumidores e comerciantes locais de Pinhal, permitindo:

- 🔍 **Buscar produtos** e comparar preços em diferentes lojas
- 💰 **Lista de compras inteligente** que calcula a melhor rota de compras
- 🏪 **Cadastro de lojas** para aumentar visibilidade
- 📊 **Preços em tempo real** atualizados constantemente

## ✨ Funcionalidades

### Para Consumidores
- ✅ Login seguro
- ✅ Busca de produtos
- ✅ Comparação de preços
- ✅ Lista de compras com cálculo de economia
- ✅ Indicador de confiabilidade

### Para Lojas
- ✅ Cadastro gratuito
- ✅ Publicação de preços
- ✅ Visibilidade aumentada
- ✅ Dashboard de gerenciamento

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Armazenamento:** localStorage (simulação de BD)
- **Autenticação:** Sistema de login customizado
- **Design:** Responsivo (Mobile, Tablet, Desktop)

## 🚀 Como Começar

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Sem necessidade de instalação

### Instalação

1. **Clone o repositório:**
```bash
git clone https://github.com/seu-usuario/economizapinhal.git
cd economizapinhal
```

2. **Abra no navegador:**
```bash
# Simplesmente abra o arquivo index.html no seu navegador
```

3. **Faça login com conta de teste:**
   - Email: `usuario@email.com`
   - Senha: `Senha123!`

## 🧪 Testando as Funcionalidades

### Login
- Acesse `login.html`

## 🔐 Segurança

⚠️ **IMPORTANTE:** Este é um projeto de demonstração/MVP.

- Senhas são armazenadas em hash simulado (NÃO é seguro para produção)
- Dados são salvos em localStorage (não persistem em servidor)
- **Para produção:** Implemente backend com Node.js, Python ou PHP
